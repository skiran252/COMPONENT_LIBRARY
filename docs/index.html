Alerts

The alert component can be used to provide information to your users such as success or error messages, but also highlighted information complementing the normal flow of paragraphs and headers on a page.

Success! This alert is a success alert.
Warning! This alert is a warning alert.
Danger! This alert is a danger alert.
Info! This alert is an info alert.

To use simply add .alert .alert-success, .alert-warning, .alert-danger or .alert-info to the element you want to be an alert.

Avatars

The avatar component is a profile image often used in combo with additional elements (e.g., a status element) to provide additional info about a user. To change the size of the component, you can either use one of the size classes or edit the font-size. The width and height of the component will adapt accordingly.

To use simply add .avatar and .avatar-sm, .avatar-md, .avatar-lg or .avatar-xl to the element you want to be an avatar or its parent.

Card

Cards are used to display content in a way that is easily digestible and readable. They can be used to display content related to a single subject or topic, or to display multiple related topics.
